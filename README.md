# ML classification- Spam detection
Developed a Spam Detection System that classifies messages as spam or non-spam (ham) using Machine Learning techniques. The project involved data preprocessing, text cleaning, tokenization, and feature extraction using techniques such as TF-IDF/Count Vectorization. Various classification algorithms were trained and evaluated to identify unwanted messages accurately. The model helps automate message filtering, improving communication efficiency and reducing exposure to spam content. Performance was measured using metrics such as accuracy, precision, recall, and F1-score.
   # Features
-Text preprocessing and cleaning
-Removal of stop words and special characters
-Feature extraction using TF-IDF Vectorization
-Machine Learning-based classification
-Accurate prediction of spam and non-spam messages
-Performance evaluation using standard metrics
  # Project Workflow
--Data Collection
-Data Cleaning and Preprocessing
-Text Vectorization using TF-IDF
-Model Training
-Model Evaluation
-Spam Prediction
# Results
The trained model successfully classifies messages into spam and ham categories with high accuracy. Performance is evaluated using metrics such as Accuracy, Precision, Recall, and F1-Score.
